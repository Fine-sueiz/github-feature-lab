# GitHub Feature Lab

这是我的 GitHub 功能练习仓库，用来熟悉仓库管理、本地同步、分支、Pull Request、Issues、Projects、Actions、Pages 和安全检查。

## 学习目标

- 跑通一次从本地修改到 GitHub 远程仓库的完整流程。
- 用 Issues 和 Projects 管理自己的编程学习任务。
- 用 Actions 尝试自动化检查和部署。
- 用 Pages 建立可以展示个人项目的入口。

## 当前进度

- Day 1: 已完成仓库创建、本地 clone、commit 和 push。
- Day 2: 正在练习分支、Pull Request 和合并流程。

## 项目方向

我希望把 GitHub 当作自己的作品集和自动化工具实验室，逐步沉淀问卷自动填写、网页翻译、视频翻译、n8n 自动化流程等项目。
