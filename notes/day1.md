# Day 1: 仓库与本地同步

## 今天要理解的概念

- Repository: GitHub 上保存项目的远程仓库。
- Clone: 把远程仓库复制到本地电脑。
- Commit: 给当前修改保存一个版本快照。
- Push: 把本地提交上传到 GitHub。
- Remote origin: 本地仓库连接的默认远程地址。
